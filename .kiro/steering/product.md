# Product Overview

KanbanGemini is a command-line interface (CLI) tool for managing a Kanban board specifically designed for AI code assistants. The tool provides a simple way to track tasks through different stages of development.

## Core Features

- Task management through Kanban columns: `id√©es`, `a_faire`, `en_cours`, `en_revision`, `termine`
- CLI-based interface for viewing and creating tasks
- JSON-based data persistence in `.kaia` file
- Task tracking with unique IDs, titles, descriptions, and acceptance criteria

## Target Use Case

This tool is designed to help AI assistants and developers manage coding tasks in a structured workflow, providing visibility into task progression from initial ideas to completion.
